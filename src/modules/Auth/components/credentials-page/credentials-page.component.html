Welcome to the signup process for Navigator

To start using Navigator you have to register with the same email you used previously.

Here's how to do it
  1. Put in the same email you used before with BFT.
  2. Choose a password you want to use.
  3. Click the register button.
  4. If you use wrong email by mistake, just repeat this process.
